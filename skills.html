USER RESEARCH

Curiosity is my guide as I seek to empathize with the user. Proto-personas, interviews, affinity diagrams and an empathy map are a few of the tools that provide structure and insight. Approaching a project with an open mind allows me to learn how to design with the most impact.

QUALITATIVE INTERVIEWS

People can tell if you genuinely care and listen to what they're sharing. If they feel safe, they will share more deeply. As a people person, this is a fulfilling part of the process for me.

USER EMPATHY

Truly understanding the user is the key to creating a design that will delight. When interviewing a foster parent for our nonprofit redesign, I learned that having a support system and mentor will make or break you. Because of that insight we designed "Support" front and center on the homepage.

FAVORITE TOOLS

Figma - This is my primary design tool. I know how to use overlays and plugins. the plugin I use the most is Contrast to check for accessibility. Creating lo, mid and high fidelity prototypes and testing them in Figma is an efficient way to get user feedback.
 
Miro - Brainstorming at the beginning of a project in Miro or Figjam is fun and easy! It's also my favorite tool to use to create an Affinity Diagram, Empathy Map and Feature Prioritization Matrix.
 
InVision - Creating and sharing Moodboards in InVision is simple and a great way to collaborate. I also like to use it to upload my wireframes and create hotspots for testing.
 
Visual Studio Code - Coding it not about computers, it's about helping people by solving problems. To be able to imagine something, start working on it, and see it take form very quickly, feels like a modern superpower!
 
GitHub - Seamless collaboration is easy with Github. I especially like the version control system to see who made changes and when they made them.

READING
